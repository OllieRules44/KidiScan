# What Is It?

Its A Firmware Dump Analysis Tool for the DX2 And DX3

# How Do I Use It?

Simply Put The analyze.py In the same folder as the App_Data.bin and Run The Command

# Whats The Command?

 RUN "python analyze.py app_data.bin" For A Basic Analysis Or "python analyze.py app_data.bin --all-strings" For A Full string dump (also saves a .strings.txt file)

 # What If I Dont Have The App_Data.bin But I Still Want To Analyse It?

 Simply Go To https://github.com/OllieRules44/Vtech-DX2-DX3-And-DX4-Watch-Files and put the App_data.bin And Download It!
